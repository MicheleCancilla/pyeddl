Creation ops¶

pyeddl.eddlT.create(shape, dev=None)¶

Create an uninitialized tensor of the specified shape.

Can also be used to create a tensor from a NumPy array: create(array) (device must be None). In this case the tensor will be initialized with values, shape, etc. from the array.

Parameters

  • shape – shape of the tensor to create

  • dev – device to use: DEV_CPU or DEV_GPU

Returns

Tensor
